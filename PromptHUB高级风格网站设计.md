# PromptHUB高级风格网站设计

## Core Features

- 提示词卡片展示

- 动态交互特效

- 提示词编辑器

- 响应式布局

- 玻璃拟态设计

## Design

现代极简主义结合Glassmorphism玻璃拟态效果，深蓝色背景配合渐变紫色和青色点缀，半透明卡片设计，丰富的动态交互效果包括悬停动画、点击涟漪、3D倾斜等特效
